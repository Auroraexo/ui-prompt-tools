# 更新日志

## 2026-02-05 - 视觉反向工程功能上线 🎨

### 新增功能

#### ✨ 视觉反向工程（一键参数回填）

**核心能力**：上传任意 UI 截图，AI 自动提取样式参数并应用到配置中。

- **后端实现**（`server/index.js`）
  - 新增 `/api/extract-ui-params` 端点
  - 使用专门优化的 Prompt 让豆包视觉模型提取 UI 参数
  - 返回结构化 JSON（圆角、间距、配色、阴影、字体、动画等）
  - 支持 markdown 代码块格式的 JSON 自动解析

- **前端实现**（`src/components/AiVisionPanel.tsx`）
  - 新增"✨ 提取参数"按钮（绿色，与"生成描述"并列）
  - 实现 `convertAIParamsToConfig` 函数，将 AI 返回的参数映射到 7 个维度的配置结构
  - 支持以下维度的自动回填：
    - 圆角 (Radius)：卡片、按钮、输入框
    - 容器 & 布局：内边距、元素间距
    - 色彩 (Color)：主色、背景色、文字透明度
    - 阴影 (Shadow)：卡片阴影、按钮阴影
    - 背景 (Glass)：毛玻璃效果
    - 字体 (Typo)：标题、正文
    - 交互 (Motion)：悬停、点击动画
  - 提取成功后在绿色提示框中显示结果和状态
  - 60 秒超时保护

- **父组件集成**（`src/App.tsx`）
  - 新增 `handleBatchUpdate` 方法支持批量更新配置
  - 通过 `onApplyConfig` prop 传递给 `AiVisionPanel`
  - 类型安全的配置合并逻辑

### 技术细节

#### 新增 API 端点

**POST** `/api/extract-ui-params`

请求体：
```json
{
  "imageUrl": "https://example.com/screenshot.png" // 或 base64 Data URL
}
```

响应体（成功）：
```json
{
  "params": {
    "圆角": { "卡片圆角": "rounded-2xl", "按钮圆角": "rounded-3xl" },
    "配色": { "主色": "indigo-500", "背景色": "zinc-950" },
    "阴影": { "卡片阴影": "shadow-[0_20px_50px_rgba(0,0,0,0.3)]" },
    // ... 其他维度
  },
  "rawText": "AI 返回的原始文本",
  "raw": { /* 完整的 API 响应 */ }
}
```

响应体（错误）：
```json
{
  "error": "错误信息"
}
```

#### 参数映射逻辑

AI 返回的 JSON 结构 → `ConfigState` 的映射关系：

| AI 返回字段 | 映射到维度 | 说明 |
|-----------|---------|-----|
| `圆角.卡片圆角` | `圆角 (Radius)` → Card | Tailwind 类名 |
| `圆角.按钮圆角` | `圆角 (Radius)` → Button | Tailwind 类名 |
| `间距.容器内边距` | `容器 & 布局` → Container | `p-*` 类名 |
| `配色.主色` | `色彩 (Color)` → Primary | `bg-*-*` 色值 |
| `阴影.卡片阴影` | `阴影 (Shadow)` → CardShadow | `shadow-[...]` |
| `背景效果.毛玻璃` | `背景 (Glass)` → GlassCard | `bg-white/5 backdrop-blur-xl` |
| `字体.标题` | `字体 (Typo)` → Heading | 字体相关类名组合 |
| `交互动画.悬停` | `交互 (Motion)` → HoverScale | 动画相关类名 |

### 使用示例

1. 启动服务：
   ```bash
   npm run dev:server  # 启动后端
   npm run dev         # 启动前端
   ```

2. 上传 UI 截图（本地文件或 URL）

3. 点击"✨ 提取参数"按钮

4. 等待 AI 分析（5-15 秒）

5. 查看左侧配置自动更新，右侧 Prompt 同步变化

6. 微调参数（可选），复制 Prompt 到 Cursor
