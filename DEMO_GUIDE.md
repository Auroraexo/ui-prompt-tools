# 🎬 视觉反向工程功能演示指南

本文档提供一个完整的演示流程，帮助你快速体验"视觉反向工程"功能。

## 🎯 演示目标

向他人展示如何通过上传 UI 截图，自动提取并应用样式参数。

## 📋 准备工作

### 1. 确保服务已启动
```bash
npm run dev:all
```

### 2. 准备演示素材

建议准备 3-5 张不同风格的 UI 截图，例如：
- 现代毛玻璃风格（如 macOS 设计）
- 扁平化设计（如 Material Design）
- 深色主题界面
- 渐变科技风

可以从以下网站获取高质量 UI 截图：
- [Dribbble](https://dribbble.com/search/ui)
- [Behance](https://www.behance.net/search/projects?search=ui%20design)
- [Mobbin](https://mobbin.com/)

## 🎭 演示流程

### 场景 1：基础功能展示

#### 步骤 1：展示默认配置
1. 打开应用（`http://localhost:5176`）
2. 指出左侧的 7 个配置维度
3. 指出右侧实时生成的 Prompt

#### 步骤 2：演示视觉反向工程
1. 在右侧"AI 图像理解"卡片找到图片上传区域
2. 上传第一张 UI 截图（推荐：毛玻璃风格）
3. **重点强调**：点击 **"✨ 提取参数"** 按钮（绿色）
4. 观察按钮状态变为"提取中..."
5. **等待结果**（5-15 秒）

#### 步骤 3：展示自动回填效果
1. 指出绿色提示框显示"✅ 已成功提取并应用 UI 参数！"
2. **滚动到左侧配置区**，逐个维度展示更新后的参数：
   - 圆角 (Radius)：`rounded-2xl` → 可能变为 `rounded-3xl`
   - 色彩 (Color)：`bg-indigo-500` → 可能变为 `bg-blue-600`
   - 阴影 (Shadow)：自动更新为截图风格
3. **滚动到右侧 Prompt 区**，展示同步更新的提示词
4. 点击"复制"按钮，展示可以直接粘贴到 Cursor

### 场景 2：不同风格对比

#### 步骤 1：上传扁平风格截图
1. 切换到扁平化设计的截图
2. 再次点击"✨ 提取参数"
3. 展示左侧配置的变化：
   - 圆角可能变小（`rounded-md`）
   - 阴影可能消失或变浅
   - 配色变为纯色

#### 步骤 2：上传深色主题截图
1. 上传深色主题截图
2. 提取参数
3. 展示配色维度的变化：
   - 背景色：`bg-zinc-950`
   - 文字透明度：`text-white/90`

### 场景 3：微调与导出

#### 步骤 1：手动微调
1. 在左侧配置中手动修改某个参数（如圆角大小）
2. 展示右侧 Prompt 实时更新

#### 步骤 2：导出 Prompt
1. 点击右侧"复制"按钮
2. 粘贴到文本编辑器展示完整 Prompt
3. **说明**：这个 Prompt 可以直接发给 Cursor 生成对应风格的代码

### 场景 4：AI 图像理解（附加功能）

#### 展示描述功能
1. 使用同一张截图
2. 在"提问文本"输入框输入："详细描述这个界面的设计风格"
3. 点击"生成描述"（蓝色按钮）
4. 展示 AI 返回的文字描述
5. **说明**：这个功能帮助理解设计理念，与参数提取互补

## 💡 演示技巧

### 强调亮点
1. **速度快**：5-15 秒完成全部 7 个维度的参数提取
2. **准确性高**：AI 能识别 Tailwind 类名
3. **可微调**：自动回填后可手动调整
4. **双功能**：既能提取参数，也能生成描述

### 对比传统方式
| 传统方式 | 视觉反向工程 |
|---------|------------|
| 观察截图 → 逐个维度手动输入 | 上传截图 → 点击按钮 → 自动回填 |
| 耗时 5-10 分钟 | 耗时 10-20 秒 |
| 容易输错类名 | AI 自动推断 Tailwind 类名 |

### 处理异常情况
如果演示中遇到错误：
1. **网络慢**：说明"通常 5-15 秒，网络慢可能需要更久"
2. **识别不准**：说明"可以手动微调，AI 提供 80% 的工作"
3. **超时**：说明"已设置 60 秒超时保护，可以稍后重试"

## 📸 演示截图建议

### 推荐使用的 UI 风格

#### 1. 毛玻璃风格（效果最好）
- 特征：半透明背景、模糊效果、大圆角
- 示例：macOS Big Sur、iOS 设计

#### 2. 扁平化风格
- 特征：纯色背景、小圆角或无圆角、无阴影
- 示例：Material Design

#### 3. 新拟物化（Neumorphism）
- 特征：柔和阴影、同色系渐变
- 示例：Dribbble 上的"neumorphism"设计

#### 4. 渐变科技风
- 特征：彩色渐变、发光效果、科技感阴影
- 示例：Web3、加密货币应用

### 避免使用的图片
- ❌ 纯文字截图（无 UI 元素）
- ❌ 低分辨率模糊图片
- ❌ 非 UI 设计（如照片、插画）
